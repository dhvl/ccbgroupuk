# CCB Group UK Ltd - Official Website

This is the new, premium Next.js website for CCB Group UK Ltd.

## Features
- **Modern Minimalist Design**: Professional UK-style aesthetic.
- **Service Focused**: Dedicated sections for Gutter Cleaning, Electrical, and Roofing.
- **Mobile Responsive**: Fully optimized for all devices.
- **Contact Form**: Integrated with Resend for professional email delivery.
- **SEO Optimized**: Built-in metadata and semantic HTML.

## Getting Started

1.  **Install Dependencies**:
    ```bash
    npm install
    ```

2.  **Environment Variables**:
    Create a `.env.local` file and add your Resend API key:
    ```env
    RESEND_API_KEY=your_resend_key_here
    ```

3.  **Run Locally**:
    ```bash
    npm run dev
    ```

4.  **Build for Production**:
    ```bash
    npm run build
    ```

## Project Structure
- `src/app`: Main pages and API routes.
- `src/components`: Reusable UI components (Header, Hero, TrustBar).
- `public/images`: Branding assets and generated photography.

## Designs
The design follows a "Navy & Red" premium palette with Inter/Montserrat typography to evoke trust and professionalism in the property maintenance sector.
